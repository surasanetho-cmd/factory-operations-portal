# FOP MVP Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans or superpowers:subagent-driven-development to implement.

**Goal:** Build Factory Operations Portal Phase 1 MVP with Supabase + Next.js 15.

**Architecture:** App Router, RLS-backed Supabase, RBAC middleware, design-system tokens.

**Tech Stack:** Next.js 15, TypeScript, Supabase, Zustand, TanStack Table, RHF+Zod, Recharts, Sonner.

---

## Task 1: Environment Setup

- Copy `.env.example` to `.env.local`
- Set `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- Run `supabase db push` or apply `supabase/migrations/001_initial_schema.sql`
- Run `supabase/seed.sql`

## Task 2: Verify Build

- Run: `npm run build`
- Run: `npm test`
- Fix any TypeScript errors

## Task 3: Create Auth Users

- Create 6 users in Supabase Auth with metadata `{ "role": "ADMIN", "name": "...", "employee_code": "..." }`
- Verify profiles auto-created via trigger

## Task 4: Smoke Test Flows

- Login as PLANNING → create plan for order
- Login as PRODUCTION → create MR → submit
- Login as STORE → approve → issue → complete
- Login as ADMIN → CSV import
