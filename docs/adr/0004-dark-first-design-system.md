# Dark-first design system over spec light UI

The original project spec called for light theme, blue primary, and glassmorphism. FOP adopts UX-UI-Design-System.md v1.0 with dark-first defaults, CSS design tokens, and Sarabun typography — better suited to factory floor environments. Shadcn UI primitives are themed via design tokens, not default Shadcn light styling.

**Status:** accepted
